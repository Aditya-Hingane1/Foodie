# 🍔 Foodie - Online Food Ordering Platform

## 📌 Overview

Foodie is a full-stack online food ordering platform that allows users to browse restaurants, explore menus, add food items to their cart, and place orders seamlessly. The platform provides a user-friendly interface for customers and an admin dashboard for managing restaurants, menu items, and orders.


## ✨ Features

### 👤 User Features

* User Registration and Login
* Secure Authentication
* Browse Restaurants
* View Food Menu
* Search Food Items
* Add Items to Cart
* Place Orders
* Order Summary
* Responsive User Interface

### 🛠 Admin Features

* Admin Login
* Manage Restaurants
* Add, Edit and Delete Food Items
* Manage Orders
* View Customer Details
* Dashboard for Monitoring Orders

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Aditya-Hingane1/Foodie.git
```

### 2. Navigate to the Project

```bash
cd Foodie
```

### 3. Install Dependencies

For Backend

```bash
npm install
```

For Frontend

```bash
cd client
npm install
```

### 4. Configure Database

* Install MySQL.
* Create a new database.
* Import the SQL file from the `database` folder.
* Update the database credentials in the backend configuration file.

### 5. Run the Backend

```bash
npm start
```

### 6. Run the Frontend

```bash
cd client
npm start
```

The application will run on:

* Frontend: `http://localhost:3000`
* Backend: `http://localhost:5000`
